# bearychat_irc
a robot bridge between bearychat and irc

## Dependences
```shell
sudo pip3 install -r reqirements.txt
```

## Usage
#### 1. cp the template
```shell
cp config.ini_example config.ini
cp bearychat.ini_example bearychat.ini
```

#### 2. just fill the template as the guid inside
